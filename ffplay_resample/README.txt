PSP Media Player ver. 0.0.5

History...
0.0.5
 - 간단한 배경화면을 불러올 수 있습니다. 480x272 크기의 jpg 파일을 프로그램 폴더에 background.jpg란 이름으로
   같이 넣어두면 됩니다. 같이 포함된 이미지는 인터넷에서 무료 이미지를 다운받아 넣어두었습니다.
 - 44100 이하의 비트레이트로 인코딩된 사운드를 리샘플링해서 정상적으로 들을 수 있습니다.
 - 클럭을 바꾸면 2초간 화면 상단에 표시해 줍니다.
 - 코덱을 아주 약간 최적화 했습니다. 그래서 눈꼽만큼 프레임레이트가 올라갔습니다. 체감하기는 어려울듯...
 - 간단하지만 프레임스킵 기능이 추가되었습니다.

0.0.4
 - 폰트를 내장방식에서 파일로딩 방식으로 바꿨습니다.
 - 영상/음악 폴더가 비어있을 경우 간단한 메시지를 보여줍니다.
 - 폰트는 터미널님의 한뷰에서 가져왔습니다.

0.0.3
 - 0.0.2 버전을 릴리즈하고 나서 테스트를 더 해봤더니 아무래도 표준클럭으로는 무리더군요... 
   그래서 동영상의 경우는 266MHz로 시작했다가 사용자가 버벅거리는 느낌을 받을 경우 삼각형 버튼을 누르면
   333MHz로 클럭을 올려 플레이하도록 수정했습니다. 올라간 클럭은 현재 재생중인 파일에만 적용됩니다.
   삼각형 버튼을 한번 더 누르면 다시 266MHz로 내려옵니다.
 - ogg 파일 포맷을 지원합니다. 새로운 기능을 하나 추가하려면 보통 라이브러리를 두개에서 세개를 포팅해야 하더군요.
   쉬운일이 아니네요.
 - 슬립모드로 들어갔다가 다시 켰을 경우 재생이 안되는 문제를 해결했습니다.
   좀 무식한 방법을 썼기 때문에 약 3초정도 걸립니다. 음악파일은 이상하게도 시간이 더 걸립니다. 원인은 불명... ㅠ.ㅠ

알고있는 문제점...

 - 여전히 무작위로 다운되는 현상은 고치지 못했습니다. malloc과 free에서 주로 죽는데 이게 gcc 라이브러리 함수라 
   어느 함수에서 이것들을 호출하는지 알 길이 없습니다.

변명...

 - 아시는 분은 알고 계시겠지만 PSP Homebrew 프로그램은 개발환경이 그다지 쾌적하지 못합니다.
   결정적으로 디버거가 없습니다. printf만으로 해결해야 하는 경우가 허다합니다.
   그래서 코덱 내부에서 문제가 발생하면 어찌할 도리가 없을 수도 있습니다.
 - 저는 시간이 그다지 넉넉하지 못합니다. 그래서 자주 업데이트하지 못할수도 있습니다.

참고...
 - 이 프로그램은 순수한 소프트웨어 디코딩이기 때문에 화질이 너무 높은 경우는 버벅거리게 됩니다.
   쾌적하게 감상하시려면 PDA기준으로 인코딩된 동영상(320x240)이 가장 좋습니다.
   풀스크린으로 PSP의 화면을 다 이용해서 보고 싶으시면 인코딩시 비트레이트를 150정도로 해주시면
   클럭을 올려서 보실 수 있습니다. 320x240 동영상 정도는 266MHz 클럭으로도 충분히 감상이 가능하실 겁니다.

읽어볼지 어떨지 장담은 못하지만 아이디어나 도움이 될만한 정보등을 알려주실 분은 아래로 메일을 보내주시기 바랍니다.
